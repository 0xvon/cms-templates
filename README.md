CMS のテンプレです

## 環境

|                    |                     |
| ------------------ | ------------------- |
| 言語               | TypeScript          |
| Web フレームワーク | React.js<br>Next.js |
| バックエンド       | microCMS            |
| CSS フレームワーク | tailwindcss         |

## セットアップ

```
$ cp .env.development.local{.sample,}
$ yarn install
$ yarn dev
```

## ビルド

```
$ yarn build
```
